# 🐢 Turtle Arcade: Two-in-One Python Games

This project features two interactive games built using Python's **Turtle Graphics** module:
1. **Etch A Sketch**: A creative drawing tool.
2. **Turtle Race**: A randomized racing game with a betting system.

---

## 🎨 Game 1: Etch A Sketch

Control the turtle to draw shapes and patterns across the screen.

### Controls:
* **W**: Move Forward
* **S**: Move Backward
* **A**: Turn Left (Counter-Clockwise)
* **D**: Turn Right (Clockwise)
* **C**: Clear the screen and return to center

---

## 🏁 Game 2: Turtle Race

Predict which of the six colorful turtles will reach the finish line first!

### How to Play:
1. **Enter Your Bet**: When the game starts, a popup will ask you: *"Which turtle will win the race? Enter a color."*
2. **Choose a Color**: Pick from `red`, `orange`, `yellow`, `green`, `blue`, or `purple`.
3. **The Race**: Turtles will line up at the start and move forward by a random distance each turn.
4. **Win/Loss**: If your chosen turtle crosses the finish line first, you win!

---

## 🛠️ Installation & Usage

### Prerequisites:
* **Python 3.x**
* The `turtle` module (standard library, no installation needed).
